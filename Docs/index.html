Research-first secure file sharing

Home-hosted sharing with delivery evidence and verifiable reachability.

SafeDrop is a local-first secure file drop that works from a home PC whenever it is online. It combines content-addressed storage, signed receipts, path-aware reachability, and explanation-first UX inside a single executable.

Core differentiators

Why SafeDrop stands out

Not just a secure transfer utility. A path-aware, evidence-oriented personal server that treats home-hosting as a publishable systems question.

🔗
Reachability

Home-hosted when online

Direct bind first, automated router mapping second, assisted connectivity third, encrypted relay last. The chosen path becomes exportable evidence.

🔒
Evidence

Proof-carrying delivery

Deterministic manifests, append-only audit events, segment receipts, and human-readable evidence cards form a useful trust layer.

⚙
Operability

Built for real hardware

Single-executable deployment, energy-aware scheduling, conservative policy defaults, and explainable admin UX.
