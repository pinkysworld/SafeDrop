
# SafeDrop IJRC Submission Checklist

- Manuscript prepared in Microsoft Word format.
- A4 page size with 20 mm margins on all sides.
- Single-column structure using standard heading styles.
- Abstract, technical sections, conclusion, and references included.
- Figures and tables numbered in sequence.
- No page numbers inserted.
- AI Usage Statement included.
- Data Availability Statement included.
- Code Availability Statement included.
- Funding Statement included.
- Conflict of Interest Declaration included.
- Ethical Considerations section included.
- References placed at the end.
- Final spell-check and grammar-check still recommended before submission.
