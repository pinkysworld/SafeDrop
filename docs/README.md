
# SafeDrop Website Deployment

This is a static website intended for GitHub Pages.

## Quick deployment

1. Create a public GitHub repository named `SafeDrop`.
2. Copy the contents of this `website/` folder into the repository root or into `docs/`.
3. In GitHub, enable **Pages** from the selected branch and folder.
4. Update links, author placeholders, and any future live-demo URLs.

## Files to customize later

- `index.html` - homepage narrative and hero text
- `research.html` - interactive 50-track catalog
- `assets/styles.css` - site design system
- `assets/tracks-data.js` - track data used by filtering UI
